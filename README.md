# RCCG New Life Assembly Website ⛪

A simple and responsive church website built to provide information about RCCG New life Assembly, including services, programs and spiritual resources.

## 🌐 Live Demo
https://tofa-dev.github.io/rccg-new-life-assembly/

---

## About the Project

This project is a church website designed to:

- Share information about church services and programs
- Provide spiritual content such as devotions and teachings
- Allow visitors to stay connected and updated
- Create an online presence for the church

## Features

- Home page with church overview
- Service schedule and programs
- Devotional / sermon content
- Gallery
- Responsive design (works on mobile & desktop)
